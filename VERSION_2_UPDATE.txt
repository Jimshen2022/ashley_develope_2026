Scenario: "I don't know where my Power BI files are"
──────────────────────────────────────────────────

1. Double-click: find_pbix_files.bat
   → Lists all .pbix files found on your system
   → Copy the path of your files

2. Double-click: config_paths.bat
   → Paste the source directory path
   → Set target directory
   → Confirm

3. Double-click: run_power_bi_converter_v2.bat
   → Menu appears
   → Select "1" for Start Conversion
   → Follow prompts in Power BI

4. Check results:
   → Power BI project files (.pbip) saved to target directory
   → Original .pbix files remain unchanged


================================================================================
🔧 TROUBLESHOOTING / 故障排除
================================================================================

Problem: "Source directory does not exist"
Solution:
  1. Run find_pbix_files.bat to locate your files
  2. Run config_paths.bat to set correct paths
  3. Try again

Problem: "Menu shows garbled characters"
Solution:
  1. Use run_power_bi_converter_v2.bat (version 2.0)
  2. Or use Convert-PowerBIFiles.ps1
  3. Or use Python: python power_bi_converter.py

Problem: "Python not found"
Solution:
  1. Install Python from https://www.python.org/downloads/
  2. Make sure to check "Add Python to PATH"
  3. Restart your computer

Problem: "Power BI not found"
Solution:
  1. Download from https://powerbi.microsoft.com/downloads/
  2. Install latest version
  3. Restart your computer

For more help, see: TROUBLESHOOTING.md
